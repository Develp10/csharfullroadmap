## How to use

1. Go in order; don't skip the foundation.
2. Build at least one pet project at each stage.
3. Keep a [weekly learning log](../README.md#weekly-learning-retrospective-template).
4. Re-check yourself against the [250+ item checklist](../checklist.md) once a month.
5. When a stage is done, mark it in your fork.
